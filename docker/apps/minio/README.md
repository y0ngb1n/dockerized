## TL;DR;

The quickest way to get started is using [docker-compose](https://docs.docker.com/compose/).

```bash
mkdir -p /usr/local/dockerized/minio ; cd $_
git svn clone https://github.com/y0ngb1n/dockerized/trunk/minio .
docker-compose up -d
```
